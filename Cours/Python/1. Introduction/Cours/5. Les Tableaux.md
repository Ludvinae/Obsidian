# <span style="color:lime">Les Tableaux (Listes) en Python</span> üêç

> En Python, on appelle **tableau** la structure **liste** (`list`).  
> Une liste est **ordonn√©e**, **modifiable** (mutable) et peut contenir **tout type d‚Äô√©l√©ments**.

---

## <span style="color:lime">1) Cr√©er un tableau</span> üß±

```python
# Litt√©raux
notes = [12, 15, 9, 17]
mixte = ["Alice", 30, True]          # possible, mais √©vitez de m√©langer les types sans raison

# √Ä partir d‚Äôun it√©rable
carres = list(range(1, 6))           # [1, 2, 3, 4, 5]
```

**Bonnes pratiques**
- Pr√©f√©rez des listes **homog√®nes** (m√™me type d‚Äô√©l√©ments) pour faciliter les traitements.
- `list()` convertit *n‚Äôimporte quel it√©rable* (range, autre liste, cha√Æne, etc.) en liste.

---

## <span style="color:lime">2) Acc√©der aux √©l√©ments</span> üîç

```python
nombres = [10, 20, 30, 40, 50]

premier  = nombres[0]    # 10
dernier  = nombres[-1]   # 50 (indice n√©gatif = depuis la fin)
milieu   = nombres[2]    # 30
```

**Indices**
- Les indices commencent √† **0**.
- Les indices **n√©gatifs** comptent depuis la fin : `-1` (dernier), `-2` (avant-dernier), etc.

## <span style="color:lime">3) Tranches (slicing)</span> ‚úÇÔ∏è

```python
nombres = [10, 20, 30, 40, 50]

sous_liste   = nombres[1:4]     # [20, 30, 40] (fin exclusive)
debut        = nombres[:3]       # [10, 20, 30]
fin          = nombres[2:]       # [30, 40, 50]
un_sur_deux  = nombres[::2]      # [10, 30, 50]
inverse      = nombres[::-1]     # [50, 40, 30, 20, 10]
```

**√Ä retenir**
- Le slicing **cr√©e une nouvelle liste** (copie superficielle).
- Syntaxe : `liste[d√©but:fin:pas]` avec `d√©but` inclus et `fin` **exclue**.

## <span style="color:lime">4) Modifier des √©l√©ments</span> ‚úèÔ∏è

```python
nombres = [10, 20, 30, 40, 50]
nombres[1] = 25                   # [10, 25, 30, 40, 50]

# Modifier une plage d‚Äôun coup
nombres[1:3] = [21, 29]           # [10, 21, 29, 40, 50]

# Remplacer par une quantit√© diff√©rente (insertion/suppression implicite)

```

## <span style="color:lime">5) Ajouter & supprimer</span> ‚ûï‚ûñ

```python
L = [10, 20, 30]

# Ajouter
L.append(40)                      # fin ‚Üí [10, 20, 30, 40]
L.insert(1, 15)                   # position 1 ‚Üí [10, 15, 20, 30, 40]
L.extend([41, 42])                # √©tendre avec une autre liste ‚Üí [10, 15, 20, 30, 40, 41, 42]

# Concat√©nation (retourne une nouvelle liste)
L2 = L + [100, 200]               # L inchang√©e, L2 nouveau tableau

# R√©p√©tition
R = [0] * 5                       # [0, 0, 0, 0, 0]

# Supprimer
L.remove(15)                      # enl√®ve la 1 ≥·µâ occurrence de 15
x = L.pop()                       # enl√®ve et retourne le dernier √©l√©ment
y = L.pop(1)                      # enl√®ve √† l‚Äôindice 1
del L[0]                          # supprime √† l‚Äôindice 0 (ne retourne rien)
L.clear()                         # vide la liste
```

**Astuce**  
`remove(valeur)` l√®ve `ValueError` si la valeur n‚Äôexiste pas. V√©rifiez avant avec `if valeur in L:`.

## <span style="color:lime">7) Trier & inverser</span> üóÇÔ∏è

```python
n = [5, 1, 9, 2]

n.sort()                          # tri EN PLACE ‚Üí [1, 2, 5, 9]
n.sort(reverse=True)              # EN PLACE d√©croissant ‚Üí [9, 5, 2, 1]

m = [5, 1, 9, 2]
m2 = sorted(m)                    # NOUVELLE liste tri√©e ‚Üí [1, 2, 5, 9] ; m inchang√©e
m3 = sorted(m, reverse=True)      # [9, 5, 2, 1]

# Trier des cha√Ænes sans tenir compte de la casse
mots = ["Z√®bre", "avion", "Banane"]
mots.sort(key=str.lower)          # ['avion', 'Banane', 'Z√®bre']

# Inverser l‚Äôordre actuel
mots.reverse()                    # EN PLACE
```

**√Ä retenir**
- `list.sort()` **modifie** la liste et retourne `None`.  
- `sorted(it√©rable)` **renvoie une nouvelle liste**.

## <span style="color:lime">8) Parcourir un tableau</span> üîÅ

```python
fruits = ["pomme", "poire", "cerise"]

# 1) Simple
for f in fruits:
    print(f)

# 2) Avec index
for i, f in enumerate(fruits):
    print(i, f)   # 0 pomme, 1 poire, 2 cerise

# 3) Tant que (rarement n√©cessaire pour lister)
i = 0
while i < len(fruits):
    print(fruits[i])
    i += 1
```

## <span style="color:lime">9) Compr√©hensions de listes</span> ‚ö°

```python
nums = [1, 2, 3, 4, 5]

carres         = [x * x for x in nums]             # [1, 4, 9, 16, 25]
pairs          = [x for x in nums if x % 2 == 0]   # [2, 4]
transforme     = [str(x) for x in nums]            # ['1','2','3','4','5']
```

**Pourquoi ?**  
Plus **compact**, souvent plus **lisible** pour filtrer/transformer.

## <span style="color:lime">11) Copie, alias & pi√®ges courants</span> üßØ

### a) Copie superficielle vs alias
```python
A = [1, 2, 3]
B = A                 # ALIAS : B et A pointent le m√™me objet
B.append(4)
# A est aussi modifi√© ‚Üí [1, 2, 3, 4]

# Copie superficielle (nouvelle liste, m√™mes √©l√©ments)
C = A[:]              # slicing
D = list(A)           # constructeur
E = A.copy()          # m√©thode
```

### b) Listes 2D : pi√®ge de la multiplication
```python
# ‚ùå Mauvais : partage les m√™mes sous-listes !
grille_mauvaise = [[0] * 3] * 4
grille_mauvaise[0][0] = 1
# Toutes les lignes changent en colonne 0

# ‚úÖ Bon : compr√©hension pour cr√©er de VRAIES sous-listes distinctes
grille = [[0] * 3 for _ in range(4)]
grille[0][0] = 1      # Seule la 1 ≥·µâ ligne change
```

### c) Copier une liste 2D
```python
# Copie superficielle ligne par ligne
copie_grille = [ligne[:] for ligne in grille]
```

## <span style="color:lime">13) Listes & cha√Ænes</span> üî°

```python
# Cha√Æne ‚Üí liste de mots
texte = "Bonjour √† tous"
mots = texte.split()                # ['Bonjour', '√†', 'tous']

# Liste de cha√Ænes ‚Üí cha√Æne

```

> `split()` produit une liste de **cha√Ænes** ; `join()` exige une liste **de cha√Ænes**.

## <span style="color:lime">14) R√©cap : m√©thodes utiles</span> üß∞

| M√©thode        | R√¥le                             |
|----------------|----------------------------------|
| `append(x)`    | ajoute `x` en fin                |
| `extend(it)`   | ajoute tous les √©l√©ments d‚Äô`it`  |
| `insert(i,x)`  | ins√®re `x` √† l‚Äôindice `i`        |
| `remove(x)`    | supprime la **1 ≥·µâ** occurrence   |
| `pop([i])`     | enl√®ve et retourne `i` (ou fin)  |
| `clear()`      | vide la liste                    |
| `index(x, s,e)`| position de `x` (entre `s` et `e`) |
| `count(x)`     | nombre d‚Äôoccurrences             |
| `sort(...)`    | tri **en place**                 |
| `reverse()`    | inverse **en place**             |
| `copy()`       | copie superficielle              |
