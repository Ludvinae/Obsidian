## <span style="color:magenta;">Pourquoi apprendre PHP ?</span> 🤔

PHP (**Hypertext Preprocessor**) est un **langage de programmation côté serveur** utilisé pour créer des sites web dynamiques et interactifs. Il alimente aujourd’hui une grande partie d’Internet — **près de 75 % des sites web** utilisent PHP d’une manière ou d’une autre (notamment WordPress, Drupal, Joomla...).

### 🔍 Avantages Clés

1. **Large adoption** : PHP bénéficie d’une immense communauté et d’un écosystème riche (librairies, frameworks, CMS).  
2. **Facilité d’apprentissage** : Sa syntaxe claire et son intégration naturelle avec le HTML le rendent accessible aux débutants.  
3. **Interopérabilité** : PHP fonctionne sur tous les systèmes d’exploitation (Windows, macOS, Linux) et s’intègre parfaitement avec des bases de données comme **MySQL**, **PostgreSQL**, **SQLite**, etc.  
4. **Open Source** : Gratuit, libre d’utilisation, et constamment mis à jour par la communauté et la **PHP Foundation**.

## <span style="color:magenta;">Installer PHP selon votre environnement</span> 🔧

Pour exécuter des scripts PHP, vous devez disposer d’un **serveur web** (comme Apache ou Nginx) et de **PHP installé localement**. Voici les différentes façons de l’installer selon votre système d’exploitation 👇

---

### 💻 **Windows**

#### Option 1 : Installation via XAMPP (recommandé pour débutants)
- Téléchargez XAMPP ici 👉 [https://www.apachefriends.org/fr/index.html](https://www.apachefriends.org/fr/index.html)  
- Installez-le : il inclut **Apache**, **MySQL**, **PHP** et **phpMyAdmin** dans un seul package.  
- Lancez **XAMPP Control Panel**, puis démarrez les modules **Apache** et **MySQL**.  
- Placez vos fichiers `.php` dans le dossier :

    ```bash
    C:\xampp\htdocs\
    ```

- Ensuite, ouvrez votre navigateur et allez à :

    ```
    http://localhost/nom_du_fichier.php
    ```

#### Option 2 : Installation manuelle
- Téléchargez la dernière version de PHP ici 👉 [https://windows.php.net/download/](https://windows.php.net/download/)  
- Décompressez le dossier dans `C:\php`  
- Ajoutez ce chemin à vos **variables d’environnement** pour pouvoir exécuter `php` depuis le terminal.  
- Vérifiez l’installation :

    ```bash
    php -v
    ```

---

### 🍏 **macOS**

#### Option 1 : Utiliser Homebrew
Homebrew est le moyen le plus simple d’installer PHP sur macOS.

```bash
brew update
brew install php
```

- Vérifiez l’installation :

    ```bash
    php -v
    ```

- Pour exécuter un serveur local dans votre dossier courant :

    ```bash
    php -S localhost:8000
    ```

Puis accédez à `http://localhost:8000` dans votre navigateur.

#### Option 2 : Installer MAMP (alternative graphique)
- Téléchargez MAMP ici 👉 [https://www.mamp.info/en/downloads/](https://www.mamp.info/en/downloads/)  
- MAMP fournit un serveur Apache, MySQL et PHP prêts à l’emploi.  
- Vos fichiers PHP doivent être placés dans :

    ```bash
    /Applications/MAMP/htdocs/
    ```

- Accédez ensuite à :

    ```
    http://localhost:8888/
    ```

---

### 🐧 **Linux (Ubuntu / Debian / Fedora)**

PHP est souvent déjà disponible dans les dépôts officiels.

#### Installation simple :
```bash
sudo apt update
sudo apt install php libapache2-mod-php php-mysql
```

#### Vérifier la version :
```bash
php -v
```

#### Lancer un serveur de développement :
```bash
php -S localhost:8000
```

#### Pour Fedora :
```bash
sudo dnf install php php-cli php-mysqlnd
```

## <span style="color:magenta;">Vérifier votre installation PHP</span> 🧪

Créez un fichier `info.php` dans votre répertoire serveur (par exemple `htdocs/`) :

```php
<?php
phpinfo();
?>
```

Ouvrez ensuite dans votre navigateur :
```
http://localhost/info.php
```

Vous verrez une page listant toutes les informations sur votre configuration PHP : version, extensions actives, serveur, etc.
